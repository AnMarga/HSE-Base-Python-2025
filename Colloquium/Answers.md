## IO. –¢–∏–ø—ã –¥–∞–Ω–Ω—ã—Ö

### –ü—Ä–æ—Å—Ç—ã–µ –≤–æ–ø—Ä–æ—Å—ã

**–í–æ–ø—Ä–æ—Å:** –ö–∞–∫–∏–µ —Å–ø–æ—Å–æ–±—ã —á—Ç–µ–Ω–∏—è –¥–∞–Ω–Ω—ã—Ö –≤—ã –∑–Ω–∞–µ—Ç–µ?

**–û—Ç–≤–µ—Ç:**

* `input()` ‚Äî —á—Ç–µ–Ω–∏–µ —Å—Ç—Ä–æ–∫–∏ –∏–∑ —Å—Ç–∞–Ω–¥–∞—Ä—Ç–Ω–æ–≥–æ –≤–≤–æ–¥–∞.
* `sys.stdin.read()` / `sys.stdin.readline()` ‚Äî —á—Ç–µ–Ω–∏–µ —á–µ—Ä–µ–∑ –º–æ–¥—É–ª—å `sys`.
* –ß—Ç–µ–Ω–∏–µ –¥–∞–Ω–Ω—ã—Ö –∏–∑ —Ñ–∞–π–ª–æ–≤: `open("file.txt")` —Å –º–µ—Ç–æ–¥–∞–º–∏ `.read()`, `.readline()`, `.readlines()`.
* –ò—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏–µ –±–∏–±–ª–∏–æ—Ç–µ–∫: `pandas.read_csv()`, `json.load()`, `csv.reader()` –∏ —Ç.–¥.

**–í–æ–ø—Ä–æ—Å:** –ö–∞–∫–∏–µ –±–∞–∑–æ–≤—ã–µ —Ç–∏–ø—ã –¥–∞–Ω–Ω—ã—Ö –≤ Python –≤—ã –∑–Ω–∞–µ—Ç–µ?

**–û—Ç–≤–µ—Ç:** `int`, `float`, `complex`, `str`, `bool`, `bytes`, `list`, `tuple`, `set`, `dict`, `NoneType`.

**–í–æ–ø—Ä–æ—Å:** –ö–∞–∫ –ø—Ä–µ–æ–±—Ä–∞–∑–æ–≤–∞—Ç—å —Å—Ç—Ä–æ–∫—É –≤ —Ü–µ–ª–æ–µ —á–∏—Å–ª–æ?

**–û—Ç–≤–µ—Ç:** –ß–µ—Ä–µ–∑ —Ñ—É–Ω–∫—Ü–∏—é `int()`. –ù–∞–ø—Ä–∏–º–µ—Ä, `int("42")` ‚Üí `42`.

**–í–æ–ø—Ä–æ—Å:** –ö–∞–∫–∏–µ –±–∏—Ç–æ–≤—ã–µ –æ–ø–µ—Ä–∞—Ü–∏–∏ –≤—ã –∑–Ω–∞–µ—Ç–µ?

**–û—Ç–≤–µ—Ç:**

* AND: `&`
* OR: `|`
* XOR: `^`
* NOT: `~`
* –°–¥–≤–∏–≥ –≤–ª–µ–≤–æ: `<<`
* –°–¥–≤–∏–≥ –≤–ø—Ä–∞–≤–æ: `>>`

**–í–æ–ø—Ä–æ—Å:** –ö–∞–∫ —Ä–∞–±–æ—Ç–∞–µ—Ç –æ–ø–µ—Ä–∞—Ç–æ—Ä `**`?

**–û—Ç–≤–µ—Ç:** –í–æ–∑–≤–µ–¥–µ–Ω–∏–µ —á–∏—Å–ª–∞ –≤ —Å—Ç–µ–ø–µ–Ω—å. –ù–∞–ø—Ä–∏–º–µ—Ä, `2 ** 3` ‚Üí `8`.

**–í–æ–ø—Ä–æ—Å:** –ö–∞–∫ —Ä–∞–±–æ—Ç–∞–µ—Ç `and` –≤ —É—Å–ª–æ–≤–∏—è—Ö?

**–û—Ç–≤–µ—Ç:**
`and` –≤–æ–∑–≤—Ä–∞—â–∞–µ—Ç –ø–µ—Ä–≤—ã–π –ª–æ–∂–Ω—ã–π —ç–ª–µ–º–µ–Ω—Ç –∏–ª–∏ –ø–æ—Å–ª–µ–¥–Ω–∏–π –∏—Å—Ç–∏–Ω–Ω—ã–π. –ü—Ä–∏–º–µ—Ä:

```python
x and y
```

–ï—Å–ª–∏ `x` –ª–æ–∂–Ω—ã–π ‚Üí —Ä–µ–∑—É–ª—å—Ç–∞—Ç `x`, –∏–Ω–∞—á–µ ‚Üí `y`.

**–í–æ–ø—Ä–æ—Å:** –ß—Ç–æ –±—É–¥–µ—Ç –≤—ã–≤–µ–¥–µ–Ω–æ –Ω–∏–∂–µ?

```python
x = 8
result = x >> 2
print(result)
```

**–û—Ç–≤–µ—Ç:** `2` (—Å–¥–≤–∏–≥ —á–∏—Å–ª–∞ 8 –≤–ø—Ä–∞–≤–æ –Ω–∞ 2 –±–∏—Ç–∞ ‚Üí 8 // 4 = 2).

**–í–æ–ø—Ä–æ—Å:** –ö–∞–∫–∏–µ –ø—Ä–µ–æ–±—Ä–∞–∑–æ–≤–∞–Ω–∏—è –∫–æ—Ä—Ä–µ–∫—Ç–Ω—ã?

```python

```

**–û—Ç–≤–µ—Ç:**

* `int("42")` ‚Äî –∫–æ—Ä—Ä–µ–∫—Ç–Ω–æ ‚Üí `42`
* `float("3.14")` ‚Äî –∫–æ—Ä—Ä–µ–∫—Ç–Ω–æ ‚Üí `3.14`
* `int("3.14")` ‚Äî –Ω–µ–∫–æ—Ä—Ä–µ–∫—Ç–Ω–æ, –≤—ã–∑–æ–≤–µ—Ç `ValueError`
* `str(123)` ‚Äî –∫–æ—Ä—Ä–µ–∫—Ç–Ω–æ ‚Üí `"123"`
* `int(342.11)` ‚Äî –∫–æ—Ä—Ä–µ–∫—Ç–Ω–æ ‚Üí `342` (–æ—Ç–±—Ä–∞—Å—ã–≤–∞–µ—Ç—Å—è –¥—Ä–æ–±–Ω–∞—è —á–∞—Å—Ç—å)

**–í–æ–ø—Ä–æ—Å:** –ß—Ç–æ –≤—ã–≤–µ–¥–µ—Ç?

```python
list(range(2,8,2))
```

**–û—Ç–≤–µ—Ç:** `[2, 4, 6]` (–¥–∏–∞–ø–∞–∑–æ–Ω –æ—Ç 2 –¥–æ 7 —Å —à–∞–≥–æ–º 2)

## –í–≤–æ–¥-–≤—ã–≤–æ–¥ —Å —Ñ–∞–π–ª–∞. –¶–∏–∫–ª—ã –∏ —É—Å–ª–æ–≤–∏—è

### –ü—Ä–æ—Å—Ç—ã–µ –≤–æ–ø—Ä–æ—Å—ã

**–í–æ–ø—Ä–æ—Å:** –ö–∞–∫–∏–µ –∫–ª—é—á–µ–≤—ã–µ —Å–ª–æ–≤–∞ —Å–≤—è–∑–∞–Ω–Ω—ã–µ —Å —Ü–∏–∫–ª–∞–º–∏ –∏ —É—Å–ª–æ–≤–∏—è–º–∏ –≤—ã –∑–Ω–∞–µ—Ç–µ?

**–û—Ç–≤–µ—Ç:**

* –£—Å–ª–æ–≤–∏—è: `if`, `elif`, `else`, `match`, `case`
* –¶–∏–∫–ª—ã: `for`, `while`, `break`, `continue`, `pass`

**–í–æ–ø—Ä–æ—Å:** –í —á–µ–º —Ä–∞–∑–Ω–∏—Ü–∞ –º–µ–∂–¥—É `if` –∏ `match`?

**–û—Ç–≤–µ—Ç:**

* `if` ‚Äî –ø—Ä–æ–≤–µ—Ä–∫–∞ —É—Å–ª–æ–≤–∏–π —Å –≤—ã—Ä–∞–∂–µ–Ω–∏—è–º–∏ True/False.
* `match` ‚Äî —Å—Ç—Ä—É–∫—Ç—É—Ä–Ω–æ–µ —Å–æ–ø–æ—Å—Ç–∞–≤–ª–µ–Ω–∏–µ —Å –æ–±—Ä–∞–∑—Ü–æ–º (pattern matching), –∞–Ω–∞–ª–æ–≥ switch-case.

**–í–æ–ø—Ä–æ—Å:** –ß—Ç–æ –±—É–¥–µ—Ç –≤—ã–≤–µ–¥–µ–Ω–æ?

```python
user = None

```

**–û—Ç–≤–µ—Ç:** `Hello Guest` (—Ç–∞–∫ –∫–∞–∫ `user` —Ä–∞–≤–µ–Ω `None`, –≤—ã—Ä–∞–∂–µ–Ω–∏–µ `user and user.get("name")` –≤–æ–∑–≤—Ä–∞—â–∞–µ—Ç `None`, –¥–∞–ª–µ–µ —Å—Ä–∞–±–∞—Ç—ã–≤–∞–µ—Ç `or "Guest"`).

**–í–æ–ø—Ä–æ—Å:** –ö–∞–∫ —Ä–∞–±–æ—Ç–∞–µ—Ç —Ü–∏–∫–ª `while`?

**–û—Ç–≤–µ—Ç:** –í—ã–ø–æ–ª–Ω—è–µ—Ç —Ç–µ–ª–æ —Ü–∏–∫–ª–∞, –ø–æ–∫–∞ —É—Å–ª–æ–≤–∏–µ –∏—Å—Ç–∏–Ω–Ω–æ. –ù–∞–ø—Ä–∏–º–µ—Ä:

```python
while condition:
    # —Ç–µ–ª–æ —Ü–∏–∫–ª–∞
```

**–í–æ–ø—Ä–æ—Å:** –ö–∞–∫ —Ä–∞–±–æ—Ç–∞–µ—Ç —Ü–∏–∫–ª `for`?

**–û—Ç–≤–µ—Ç:** –ò—Ç–µ—Ä–∏—Ä—É–µ—Ç –ø–æ –∫–æ–ª–ª–µ–∫—Ü–∏–∏ –∏–ª–∏ –≥–µ–Ω–µ—Ä–∞—Ç–æ—Ä—É –∑–Ω–∞—á–µ–Ω–∏–π. –ù–∞–ø—Ä–∏–º–µ—Ä:

```python
for x in iterable:
    # —Ç–µ–ª–æ —Ü–∏–∫–ª–∞
```

### –°—Ä–µ–¥–Ω–∏–µ –≤–æ–ø—Ä–æ—Å—ã

**–û—Ç–≤–µ—Ç:** –ü—Ä–∏–Ω—É–¥–∏—Ç–µ–ª—å–Ω–æ —Å–±—Ä–∞—Å—ã–≤–∞–µ—Ç –±—É—Ñ–µ—Ä –Ω–∞ –¥–∏—Å–∫. –ù–∞–ø—Ä–∏–º–µ—Ä, `file.flush()` –≥–∞—Ä–∞–Ω—Ç–∏—Ä—É–µ—Ç –∑–∞–ø–∏—Å—å –≤—Å–µ—Ö –¥–∞–Ω–Ω—ã—Ö, –¥–∞–∂–µ –µ—Å–ª–∏ –±—É—Ñ–µ—Ä –µ—â—ë –Ω–µ –ø–æ–ª–Ω—ã–π.

**–í–æ–ø—Ä–æ—Å:** –ß—Ç–æ –±—É–¥–µ—Ç –≤—ã–≤–µ–¥–µ–Ω–æ –Ω–∏–∂–µ?

```python
lists = [["üê∏", "üå∏"], ["üê∏", "üå∏", "üåπ"], None]
```

**–û—Ç–≤–µ—Ç:**

* –ü–µ—Ä–≤—ã–π —Å–ø–∏—Å–æ–∫ —Å–æ–æ—Ç–≤–µ—Ç—Å—Ç–≤—É–µ—Ç —à–∞–±–ª–æ–Ω—É `["üê∏", ("üåπ"|...)]`

### –°–ª–æ–∂–Ω—ã–µ –≤–æ–ø—Ä–æ—Å—ã

**–í–æ–ø—Ä–æ—Å:** –ö–∞–∫ –ø—Ä–∞–≤–∏–ª—å–Ω–æ —á–∏—Ç–∞—Ç—å –¥–∞–Ω–Ω—ã–µ –∏–∑ —Ñ–∞–π–ª–∞, –µ—Å–ª–∏ –æ–±—ä—ë–º —Ñ–∞–π–ª–∞ - –Ω–µ—Å–∫–æ–ª—å–∫–æ —Ç–µ—Ä–∞–±–∞–π—Ç?

**–û—Ç–≤–µ—Ç:** –ò—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å –ø–æ—ç—Ç–∞–ø–Ω–æ–µ —á—Ç–µ–Ω–∏–µ –∏ –æ–±—Ä–∞–±–æ—Ç–∫—É –±–µ–∑ –∑–∞–≥—Ä—É–∑–∫–∏ –≤—Å–µ–≥–æ —Ñ–∞–π–ª–∞ –≤ –ø–∞–º—è—Ç—å:

## –ö–æ–Ω—Ç–µ–π–Ω–µ—Ä—ã

### –ü—Ä–æ—Å—Ç—ã–µ –≤–æ–ø—Ä–æ—Å—ã

**–í–æ–ø—Ä–æ—Å:** –ß—Ç–æ —Ç–∞–∫–æ–µ –º–Ω–æ–∂–µ—Å—Ç–≤–æ?

**–û—Ç–≤–µ—Ç:** –ú–Ω–æ–∂–µ—Å—Ç–≤–æ (`set`) ‚Äî –Ω–µ—É–ø–æ—Ä—è–¥–æ—á–µ–Ω–Ω–∞—è –∫–æ–ª–ª–µ–∫—Ü–∏—è —É–Ω–∏–∫–∞–ª—å–Ω—ã—Ö —ç–ª–µ–º–µ–Ω—Ç–æ–≤. –ü—Ä–∏–º–µ—Ä: `s = {1, 2, 3}`.

**–í–æ–ø—Ä–æ—Å:** –ß—Ç–æ —Ç–∞–∫–æ–µ —Å–ª–æ–≤–∞—Ä—å?

**–û—Ç–≤–µ—Ç:** –°–ª–æ–≤–∞—Ä—å (`dict`) ‚Äî –∫–æ–ª–ª–µ–∫—Ü–∏—è –ø–∞—Ä "–∫–ª—é—á: –∑–Ω–∞—á–µ–Ω–∏–µ" —Å —É–Ω–∏–∫–∞–ª—å–Ω—ã–º–∏ –∫–ª—é—á–∞–º–∏. –ü—Ä–∏–º–µ—Ä: `d = {"a": 1, "b": 2}`.

**–í–æ–ø—Ä–æ—Å:** –ö–∞–∫–æ–µ –ø—Ä–∏–º–µ–Ω–µ–Ω–∏–µ –º–Ω–æ–∂–µ—Å—Ç–≤ –∏ —Å–ª–æ–≤–∞—Ä–µ–π –≤—ã –∑–Ω–∞–µ—Ç–µ?

**–û—Ç–≤–µ—Ç:**

* –ú–Ω–æ–∂–µ—Å—Ç–≤–∞: —É–¥–∞–ª–µ–Ω–∏–µ –¥—É–±–ª–∏–∫–∞—Ç–æ–≤, –æ–ø–µ—Ä–∞—Ü–∏–∏ –æ–±—ä–µ–¥–∏–Ω–µ–Ω–∏—è, –ø–µ—Ä–µ—Å–µ—á–µ–Ω–∏—è, —Ä–∞–∑–Ω–æ—Å—Ç–∏.
* –°–ª–æ–≤–∞—Ä–∏: —Ö—Ä–∞–Ω–µ–Ω–∏–µ –¥–∞–Ω–Ω—ã—Ö —Å –±—ã—Å—Ç—Ä—ã–º –¥–æ—Å—Ç—É–ø–æ–º –ø–æ –∫–ª—é—á—É, –∫—ç—à–∏—Ä–æ–≤–∞–Ω–∏–µ, —Å—Ç—Ä—É–∫—Ç—É—Ä—ã —Ç–∏–ø–∞ JSON.

**–í–æ–ø—Ä–æ—Å:** –ß—Ç–æ –¥–µ–ª–∞–µ—Ç —ç—Ç–æ—Ç –∫–æ–¥?

```python
{x**2 for x in range(10) if x % 2 == 0}
```

**–û—Ç–≤–µ—Ç:** –°–æ–∑–¥–∞–µ—Ç –º–Ω–æ–∂–µ—Å—Ç–≤–æ –∫–≤–∞–¥—Ä–∞—Ç–æ–≤ —á–µ—Ç–Ω—ã—Ö —á–∏—Å–µ–ª –æ—Ç 0 –¥–æ 9: `{0, 4, 16, 36, 64}`.

**–í–æ–ø—Ä–æ—Å:** –ö–∞–∫–æ–π –æ–±—ä–µ–∫—Ç —è–≤–ª—è–µ—Ç—Å—è —Ö–µ—à–∏—Ä—É–µ–º—ã–º?

**–û—Ç–≤–µ—Ç:** –û–±—ä–µ–∫—Ç, —É –∫–æ—Ç–æ—Ä–æ–≥–æ –Ω–µ–∏–∑–º–µ–Ω—è–µ–º–æ–µ —Å–æ—Å—Ç–æ—è–Ω–∏–µ –∏ –µ—Å—Ç—å –º–µ—Ç–æ–¥ `__hash__()`. –ü—Ä–∏–º–µ—Ä—ã: `int`, `float`, `str`, `tuple` (–µ—Å–ª–∏ –≤—Å–µ —ç–ª–µ–º–µ–Ω—Ç—ã —Ö–µ—à–∏—Ä—É–µ–º—ã), `frozenset`.

**–í–æ–ø—Ä–æ—Å:** –ß—Ç–æ –º–æ–∂–µ—Ç –±—ã—Ç—å –∫–ª—é—á–æ–º –≤ —Å–ª–æ–≤–∞—Ä–µ?

**–û—Ç–≤–µ—Ç:** –õ—é–±–æ–π —Ö–µ—à–∏—Ä—É–µ–º—ã–π –æ–±—ä–µ–∫—Ç, –Ω–∞–ø—Ä–∏–º–µ—Ä: —á–∏—Å–ª–∞, —Å—Ç—Ä–æ–∫–∏, –∫–æ—Ä—Ç–µ–∂–∏ —Å —Ö–µ—à–∏—Ä—É–µ–º—ã–º–∏ —ç–ª–µ–º–µ–Ω—Ç–∞–º–∏, `frozenset`.

**–í–æ–ø—Ä–æ—Å:** –ß—Ç–æ –º–æ–∂–µ—Ç –±—ã—Ç—å –∑–Ω–∞—á–µ–Ω–∏–µ–º –≤ —Å–ª–æ–≤–∞—Ä–µ?

**–û—Ç–≤–µ—Ç:** –õ—é–±–æ–π –æ–±—ä–µ–∫—Ç Python, –≤–∫–ª—é—á–∞—è –∏–∑–º–µ–Ω—è–µ–º—ã–µ –∏ –Ω–µ–∏–∑–º–µ–Ω—è–µ–º—ã–µ —Ç–∏–ø—ã.

**–í–æ–ø—Ä–æ—Å:** –ß—Ç–æ —Ç–∞–∫–æ–µ —Ö–µ—à?

**–û—Ç–≤–µ—Ç:** –ß–∏—Å–ª–æ–≤–æ–µ –∑–Ω–∞—á–µ–Ω–∏–µ, –∫–æ—Ç–æ—Ä–æ–µ —É–Ω–∏–∫–∞–ª—å–Ω–æ (–∏–ª–∏ –ø–æ—á—Ç–∏ —É–Ω–∏–∫–∞–ª—å–Ω–æ) –∏–¥–µ–Ω—Ç–∏—Ñ–∏—Ü–∏—Ä—É–µ—Ç –æ–±—ä–µ–∫—Ç –¥–ª—è –±—ã—Å—Ç—Ä–æ–≥–æ –ø–æ–∏—Å–∫–∞ –≤ —Å—Ç—Ä—É–∫—Ç—É—Ä–∞—Ö —Ç–∏–ø–∞ `dict` –∏–ª–∏ `set`.

### –°—Ä–µ–¥–Ω–∏–µ –≤–æ–ø—Ä–æ—Å—ã

**–í–æ–ø—Ä–æ—Å:** –ö–∞–∫ –±–µ—Ä–µ—Ç—Å—è —Ö–µ—à —É –∫–æ—Ä—Ç–µ–∂–∞?

**–û—Ç–≤–µ—Ç:** –ß–µ—Ä–µ–∑ –∫–æ–º–±–∏–Ω–∞—Ü–∏—é —Ö–µ—à–µ–π –≤—Å–µ—Ö —ç–ª–µ–º–µ–Ω—Ç–æ–≤ –∫–æ—Ä—Ç–µ–∂–∞. –ö–æ—Ä—Ç–µ–∂ —Ö–µ—à–∏—Ä—É–µ–º —Ç–æ–ª—å–∫–æ –µ—Å–ª–∏ –≤—Å–µ –µ–≥–æ —ç–ª–µ–º–µ–Ω—Ç—ã —Ö–µ—à–∏—Ä—É–µ–º—ã.

**–í–æ–ø—Ä–æ—Å:** –ß—Ç–æ –≤—ã –∑–Ω–∞–µ—Ç–µ –æ –º–µ—Ç–æ–¥–∞—Ö `keys()`, `values()`, `items()` –≤ —Å–ª–æ–≤–∞—Ä–µ? –ö–∞–∫–æ–π —Ç–∏–ø —É –≤–æ–∑–≤—Ä–∞—â–∞–µ–º—ã—Ö –∑–Ω–∞—á–µ–Ω–∏–π?

**–û—Ç–≤–µ—Ç:**

* `keys()` ‚Äî –≤–æ–∑–≤—Ä–∞—â–∞–µ—Ç –æ–±—ä–µ–∫—Ç `dict_keys` (–∏—Ç–µ—Ä–∏—Ä—É–µ–º—ã–π).
